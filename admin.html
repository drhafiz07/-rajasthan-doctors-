Enable Write Access

To allow the admin panel to add, edit and delete rows, you need to deploy a free Google Apps Script as a web app. It takes about 5 minutes.

  1. Open your Google Sheet → click Extensions → Apps Script
  2. Delete any existing code and paste the script from the apps-script.gs file (download below)
  3. Click Deploy → New deployment → Web app
  4. Set "Execute as: Me" and "Who has access: Anyone"
  5. Click Deploy → copy the Web App URL
  6. Paste the URL into the APPS_SCRIPT_URL field at the top of admin.html
📄 The apps-script.gs file is included in your download package.
